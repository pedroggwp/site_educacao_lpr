Estruturas de Repetição

As estruturas de repetição são utilizadas quando queremos que um bloco de código seja executado várias vezes, basicamente como funciona um loop.

    Em Python existem duas formas de criar uma estrutura de repetição:

  • O for é usado quando se quer iterar sobre um bloco de código um número determinado de vezes.
    • for a in range(0,5):
      print(a)
  • O for também pode ser utilizado para percorrer uma sequência de dados (seja esse uma lista, uma tupla, uma string), executando um bloco de código.
    • frutas = ["maçã", "banana", "cereja"]
    # Percorrendo a lista
    for fruta in frutas:
      print(fruta)
  • Às vezes, você pode precisar do índice do elemento além do valor. Isso pode ser feito usando a função enumerate().
    • # Exemplo de lista
    frutas = ["maçã", "banana", "cereja"]
    # Percorrendo a lista com índices
    for indice, fruta in enumerate(frutas):
      print(f"Índice: {indice}, Fruta: {fruta}")
  • O while é usado quando queremos que o bloco de código seja repetido até que uma condição seja satisfeita. Ou seja, é necessário que uma expressão booliana dada seja verdadeira. Assim que ela se tornar falsa, o while para.
    • numeroRepetir = 0
    while numeroRepetir < 10:
      print(numeroRepetir)
      numeroRepetir += 1
  • O while True cria um loop que continua para sempre, até que você o pare usando a instrução break. Esse tipo de loop é útil quando você quer que o programa continue rodando até que algo específico aconteça.
    • numeroRepetir = 100
    while True:
      numeroRepetir = numeroRepetir + 1
      if numeroRepetir > 100:
        break
