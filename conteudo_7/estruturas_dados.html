Estruturas de dados

Em Python, existem várias estruturas de dados que podem ser usadas para resolver diferentes tipos de problemas e são úteis em muitas situações. As principais estruturas incluem Listas, Sets, Dicionários e Tuplas. Vamos explorar suas diferenças e principais características.

Listas

Uma lista é a estrutura de dados mais básica do Python e armazena os dados em sequência, onde cada elemento possui sua posição na lista, denominada de índice. O primeiro elemento é sempre o índice zero e a cada elemento inserido na lista esse valor é incrementado.

No Python, uma lista pode armazenar qualquer tipo de dado primitivo (string, inteiro, float, etc). Veja como declaramos uma lista vazia e uma com valores:

lista_vazia = [] lista_com_valores = [1, 2, 3]

Tuplas

Uma tupla é uma estrutura bastante similar a uma lista, com apenas uma diferença: os elementos inseridos em uma tupla não podem ser alterados, diferente de uma lista onde podem ser alterados livremente. Sendo assim, em um primeiro momento, podemos pensar em uma tupla como uma lista que não pode ser alterada.

É certo que as tuplas possuem diversas características das listas, porém os usos são distintos. As listas são destinadas a serem sequências homogêneas, enquanto que as Tuplas são estruturas de dados heterogêneas. Veja como declaramos uma tupla abaixo:

tupla_inteiros = (1, 2, 3) tupla_heterogeneos = (1,"Oi",3)

Dicionários

No Python, os dicionários são coleções de itens desordenados com uma diferença bem grande quando comparados às outras coleções (lists, sets, tuples, etc): um elemento dentro de um dicionário possui uma chave atrelada a ele, uma espécie de identificador. Sendo assim, é muito utilizado quando queremos armazenar dados de forma organizada e que possuem identificação única (como acontece em bancos de dados).

Veja como declaramos dicionários abaixo:

dicionario_com_inteiros_chave = {1: "Henrique", 2: "Juan"} dicionario_con_strings_chave = {"nome": "Juan", "sobrenome": "Almeida"}

Veja um exemplo prático

Esse será um exemplo prático de inserção em uma lista e como ela se comporta.

Adicionar um item na lista

Quer saber os métodos mais importantes de uma lista? Clique aqui!

Append

Adiciona um item ao fim da lista. Sintaxe:

list.append(x)

Insert

Insere um item em uma dada posição. O primeiro argumento é o índice do elemento antes do qual será feita a inserção. Veja a sintaxe abaixo:

list.insert(i, x)

Remove

Remove o primeiro item encontrado na lista cujo valor é igual a x. Se não existir valor igual, uma exceção ValueError é levantada. Veja a sintaxe abaixo:

list.remove(x)

Pop

Remove o item na posição fornecida na lista e retorna. Se nenhum índice for especificado, a.pop() remove e retorna o último item da lista. Levanta um IndexError se a lista estiver vazia ou o índice estiver fora do intervalo da lista. Veja a sintaxe abaixo:

list.pop(i)

Clear

Remove todos os itens de uma lista. Veja a sintaxe abaixo:

list.clear()

Count

Devolve o número de vezes em que x aparece na lista. Veja a sintaxe abaixo:

list.count(x)

Sort

Ordena os itens na lista (os argumentos podem ser usados para personalizar a ordenação, veja a função sorted() para maiores explicações). Veja a sintaxe abaixo

list.sort(*, key=None, reverse=False)

Reverse

Inverte a ordem dos elementos na lista. Veja a sintaxe abaixo

list.reverse()
