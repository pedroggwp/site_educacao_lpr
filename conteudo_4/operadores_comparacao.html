Operadores de Comparação

Os operadores de comparação são usados para comparar dois valores e retornam um valor booleano (verdadeiro ou falso) com base na comparação.

Operador ==

Verifica se dois valores são iguais

x = 5 y = 5 resultado = (x == y) print(resultado) # Output: True

Operador !=

Verifica se dois valores são diferentes

x = 5 y = 5 resultado = (x != y) print(resultado) # Output: True

Operador >

Verifica se o valor do lado esquerdo é maior que o valor do lado direito

x = 5 y = 5 resultado = (y > x) print(resultado) # Output: False

Operador <

Verifica se o valor do lado esquerdo é menor que o valor do lado direito

x = 5 y = 5 resultado = (x < y) print(resultado) # Output: False

Operador >=

Verifica se o valor do lado esquerdo é maior ou igual ao valor do lado direito

x = 5 y = 5 resultado = (x >= y) print(resultado) # Output: True

Operador <=

Verifica se o valor do lado esquerdo é menor ou igual ao valor do lado direito

x = 5 y = 5 resultado = (x <= y) print(resultado) # Output: True
