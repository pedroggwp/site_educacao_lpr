Operadores

Os operadores em Python são divididos em várias categorias principais, como: Operadores Aritméticos, Operadores de Comparação, Operadores Lógicos, Operadores de Atribuição, e etc.

Operadores Aritméticos

Os operadores aritméticos são usados para realizar operações matemáticas em Python, como adição, subtração, multiplicação, divisão, entre outras.

Operador de adição (+)

Utilizado para realizar a soma de dois valores

i = 7 j = 3 print(i + j) # Output: 10

Operador de subtração (-)

Utilizado para realizar a subtração de dois valores

i = 7 j = 3 print(i - j) # Output: 4

Operador de multiplicação (*)

Utilizado para realizar a multiplicação de dois valores

i = 7 j = 3 print(i * j) # Output: 21

Operador de divisão (/)

Utilizado para realizar a divisão de dois valores

i = 7 j = 3 print(i / j) # Output: 2.3333333333333335

Operador de potenciação (**)

Utilizado para elevar um número a uma determinada potência

i = 7 j = 3 print(i ** j) # Output: 343

Operador de resto da divisão (%)

Utilizado para obter o resto da divisão entre dois números

i = 7 j = 3 print(i % j) # Output: 1

Operador de divisão inteira (//)

Utilizado para obter a parte inteira da divisão entre dois números

i = 7 j = 3 print(i // j) # Output: 2
