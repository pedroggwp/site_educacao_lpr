Operadores Lógicos

Operadores lógicos são usados para combinar expressões booleanas e retornar um valor booleano. Para compreender os operadores lógicos em Python, é fundamental entender como funcionam as estruturas condicionais, pois os operadores lógicos são frequentemente usados para combinar múltiplas condições em instruções if, elif e while. Eles nos permitem criar expressões complexas que nos ajudam a controlar o fluxo do programa de acordo com diferentes cenários.

Operador and

Retorna True se ambas as expressões forem verdadeiras

a = True b = False print(a and b) # Output: False
a = 5 b = 10 print(a < 10 and b > 5) # Output: True

Operador or

O operador or retorna True se pelo menos uma das expressões for verdadeira. Caso ambas sejam falsas, retorna False.

a = True b = False print(a or b) # Output: True
a = 5 b = 10 print(a < 10 or b > 5) # Output: True

Operador not

O operador not inverte o valor lógico de uma expressão. Se a expressão for True, retorna False, e vice-versa.

a = True print(not a) # Output: False
a = 5 print(not a < 10) # Output: False

Exemplos em Condicionais

Os operadores lógicos são frequentemente usados em condicionais para combinar múltiplas condições.

Exemplo com if e and

a = 5
b = 10
if a < 10 and b > 5:
    print("Ambas as condições são verdadeiras.")
# Output: Ambas as condições são verdadeiras.

Exemplo com if e or

a = 5
b = 3
if a < 10 or b > 5:
    print("Pelo menos uma das condições é verdadeira.")
# Output: Pelo menos uma das condições é verdadeira.

Exemplo com if e not

a = 5
if not a > 10:
    print("A condição é falsa, então o not a torna verdadeira.")
# Output: A condição é falsa, então o not a torna verdadeira.

Precedência de Operadores Lógicos

A precedência dos operadores lógicos é a ordem em que as operações são realizadas quando múltiplos operadores são usados em uma expressão. Em Python, a precedência é a seguinte (da mais alta para a mais baixa):

  • not
  • and
  • or
